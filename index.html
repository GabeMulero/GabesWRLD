About

I’m an individual programmer building this for personal finance automation only—balancing accounts, transactions, and related workflows for myself. It is not a product, company, or public service.

This project is non-commercial: it is not offered for sale, not run for profit, and not intended to be monetized, licensed, or otherwise capitalized on in any way. The code connects to Plaid for my own use in a controlled setup.

Technically, it is a small development / integration layer: Plaid APIs exposed to MCP-compatible clients (for example AI assistants) for personal testing. End-user bank sign-in runs through Plaid Link and Plaid’s systems. This page exists only to describe the project honestly for registration and transparency.

Privacy

This static page does not set tracking cookies or collect personal information. Plaid’s services handle bank connection and data according to Plaid’s terms and policies. For questions about the MCP server, see the plaid-mcp-python repository.
