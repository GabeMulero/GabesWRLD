Plaid MCP

A Python Model Context Protocol server for Plaid — accounts, balances, and transactions.

This site (GitHub) MCP server code Privacy

About

This is a development and integration project: it exposes Plaid APIs to MCP-compatible clients (for example AI assistants) so you can build and test account linking and financial data flows in a controlled way.

End-user financial authentication runs through Plaid Link and Plaid’s infrastructure; this site is only a project description for integrators and partners.

Privacy

This static page does not set tracking cookies or collect personal information. Plaid’s services handle bank connection and data according to Plaid’s terms and policies. For questions about the MCP server, see the plaid-mcp-python repository.
